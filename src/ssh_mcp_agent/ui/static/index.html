How to use SSH MCP Agent

1. Configure Hosts

Click "Manage Hosts" to add your SSH servers. You can save multiple configurations with different credentials.

2. Select a Model

Enter the name of an Ollama model (e.g., llama3.2). Ensure the model is pulled in Ollama.

3. Natural Language Queries

Ask the agent to perform tasks like:

  • "What is the CPU usage on the server?"
  • "List files in /var/log"
  • "Download /etc/hosts to my local machine"

The "Execution Logs" pane on the right shows exactly which tools the agent is calling and the raw data it receives.
